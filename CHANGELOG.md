### Added

- Responsive design for mobile, tablet, and desktop
- Hero section with call-to-action buttons
- About section with feature cards
- Services section with 4 main services
- Comprehensive pricing section:
  - Trị liệu thư giãn – giảm mệt mỏi
  - Trị liệu xương khớp – giảm đau
  - Trị liệu chuyên sâu (đá nóng, thảo dược)

  - Gói liệu trình & ưu đãi

- Footer with quick links
- Smooth scroll navigation
- Fade-in animations on scroll
- Parallax effect on hero section
- Ripple effect on buttons
- Counter animation for prices
- Hover effects on cards and items

### Project Structure
- Modular CSS architecture:
  - `variables.css` - CSS variables
  - `base.css` - Base styles
  - `header.css` - Header styles
  - `hero.css` - Hero section
  - `sections.css` - Main content sections
  - `footer.css` - Footer styles
  - `animations.css` - Keyframe animations
  - `responsive.css` - Media queries
- Modular JavaScript:
  - `smooth-scroll.js` - Smooth scrolling
  - `animations.js` - Scroll animations
  - `interactions.js` - User interactions
  - `main.js` - Entry point
